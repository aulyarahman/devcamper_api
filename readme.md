# DevCamper API

> Backend API for Devcamper application whic is a bootcamp directory

## Usage

Rename "config/config.env.env to "config/config.env and update the values/settings to your on

## Install Depencies

```
npm install
```

## Run app

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
